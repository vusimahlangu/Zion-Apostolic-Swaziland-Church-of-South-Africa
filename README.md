## Quick Start
1. **Double-click un.bat** OR
2. **Run in PowerShell:** python app.py
3. **Visit:** http://localhost:5000
## Features
- Church information dashboard
- Branch management (9 provinces)
- Member registration system
- Youth foundation programs
- REST API for mobile apps
## API Endpoints
- GET / - Church website
- GET /api/health - System status
- GET /api/church/info - Church details
- GET /api/branches - All branches
